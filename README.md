# ai-agent-portfolio
이 저장소는 LLM, RAG, AI Agent 기반의
서비스형 AI 프로젝트를 기획, 실험, 구현한 과정을 정리한 개인 포트폴리오입니다.

## Portfolio Roadmap
- Project 1: Data-based Problem Definition & Prediction
- Project 2: AI Service Implementation (API + UI)
- Project 3: RAG-based Domain AI Agent
- Project 4: Real-time AI Agent with Unreal Engine
- Project 5: Multi-Agent Decision System

각 프로젝트는 문제 정의 → 접근 방식 → 한계 → 개선 방향을 중심으로 정리되어 있습니다.
